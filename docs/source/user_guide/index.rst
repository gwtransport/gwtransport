User Guide
==========

This guide covers the fundamental concepts and assumptions for using ``gwtransport`` to analyze groundwater transport of solutes and temperature. It is recommended to read the :doc:`concepts` page first to understand the core modeling approach, and then consult :doc:`assumptions` to determine when ``gwtransport`` is appropriate for your application.
